# BusReservationSystem
A bus reservation system coded in c using turbo c, this is the first major project I worked on with ms. Gatus, at that time for our final requirement in our Laboratory

##Tools used
-TurboC++
-VSC 
-C

##Features
-Choosing your own bus
-Reserving a bus seat
-Displaying available seat
-Cancelling a reservation
-Displaying all reservations
